# 🐕 Dogfy Life
Simulador interactivo de mascota en la terminal: cuida, alimenta y juega con tu perro.

## ✨ Características
- Simulador de mascota virtual estilo Tamagotchi con un Golden Retriever en arte ASCII
- Animaciones del perro según su estado: feliz, comiendo, durmiendo, triste, paseando, jugando
- Pantallas de inicio, comida, actividades, minijuegos y estadísticas
- Actividades: paseo, jugar, dormir y caricias
- Minijuegos integrados (atrapar premios "catch" y memoria "memory")
- Guardado de progreso en `dogfy_save.json`
- Temática de Dogfy Diet (comida natural para perros)
- Solo librería estándar de Python (`curses`)

## 🚀 Cómo jugar / ejecutar
```bash
python3 dogfy_life.py
```
Requiere Python 3.8+ y una terminal con soporte Unicode.

## 🎮 Controles
| Tecla | Acción |
| --- | --- |
| `↑` `↓` / `W` `S` | Navegar opciones |
| `←` `→` / `A` `D` | Mover en el minijuego de atrapar |
| `Enter` / `Espacio` | Confirmar |
| `1` `2` `3` `4` | Acciones rápidas / menús |
| `H` | Ayuda |
| `Q` / `Esc` | Salir / volver |

## 🛠️ Tecnología
- Python 3.8+
- Librería `curses` (terminal)
- Arte ASCII y animaciones por frames
